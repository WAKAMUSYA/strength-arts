# Strength Arts

科学、研究、身体トレーニング、感覚認識、そして哲学を通じて、複数の視点から強さを探求し発展させるウェブサイトです。

## テクノロジースタック

- **フレームワーク**: Next.js 14 (App Router)
- **スタイリング**: Tailwind CSS
- **言語**: TypeScript

## プロジェクト構成

```
├── app/
│   ├── layout.tsx          # ルートレイアウト
│   ├── page.tsx            # メインページ
│   └── globals.css         # グローバルスタイル
├── components/
│   ├── HeroSection.tsx     # ヒーローバナー
│   ├── AboutSection.tsx    # アバウトコンテンツ
│   ├── CategoriesSection.tsx # 4つのカテゴリーカード
│   ├── ContentSection.tsx  # プラットフォーム（たむ研、OSA）
│   ├── CTASection.tsx      # コールトゥアクション
│   └── Footer.tsx          # フッター
├── package.json            # 依存関係
├── tailwind.config.js      # Tailwind設定
├── tsconfig.json           # TypeScript設定
└── next.config.js          # Next.js設定
```

## インストール方法

1. 依存関係をインストール：
```bash
npm install
```

2. 開発サーバーを実行：
```bash
npm run dev
```

3. ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

## 本番ビルド

```bash
npm run build
npm run start
```

## 機能

- **レスポンシブデザイン**: モバイルファーストで、あらゆる画面サイズに対応
- **ダークテーマ**: rpstrength.comにインスパイアされた専門的なダークデザイン
- **コンポーネント化**: 再利用可能で保守しやすいコンポーネント構造
- **パフォーマンス**: Next.js最適化設定
- **アクセシビリティ**: セマンティックHTMLとARIA属性

## セクション

### 1. ヒーローセクション
- 力強いタイトルとメッセージング
- 明確な価値提案
- アクションボタン

### 2. アバウトセクション
- Strength Arts の哲学を説明
- 強さの4つの次元をハイライト

### 3. カテゴリーセクション（グリッドレイアウト）
- **ボディメイク**: 脂肪減少、筋肉増加、体型開発
- **アスリートトレーニング**: パフォーマンス、スピード、強さ、競技
- **ムーブメントマスタリー**: 効率的な動き、力の伝達、非衝突ダイナミクス
- **トレーナー教育**: コーチングスキル、認定試験準備

### 4. コンテンツセクション（プラットフォーム）
- **たむ研**: 実験的で研究ベースのコンテンツ
- **OSA (Origin Strength Arts)**: 身体、認識、哲学の深い探求

### 5. CTA セクション
- ユーザーエンゲージメントを促進
- 目立つコールトゥアクション

### 6. フッター
- ナビゲーションリンク
- ソーシャルメディア
- 法的リンク

## カスタマイズ

### 色
`tailwind.config.js` を編集してカラースキームをカスタマイズします。

### タイポグラフィ
`app/globals.css` を変更してフォント関連の変更を行います。

### コンテンツ
関連する `.tsx` ファイルでコンポーネントのコンテンツを更新します。

## 注記

- "武術的身体操作"（格闘技風の身体制御）は、英語でよりよく概念を捉えるために **"Movement Mastery"** に改名されました。
- デザインは明確性、力強さ、プロフェッショナリズムを強調し、クラッターのない施しに重点を置いています。
